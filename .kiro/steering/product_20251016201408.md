# Product Overview

Azure ISP Monitor is a serverless internet connectivity monitoring system that sends email alerts when internet connection goes down.

## Core Functionality

- Lightweight Python heartbeat agent runs on monitored device (e.g., home computer)
- Agent sends periodic HTTP pings to Azure Function endpoint
- Azure Monitor detects missing pings and triggers email alerts
- Designed for low-cost operation (~$2-10/month on Azure free tier)

## Key Components

1. **Heartbeat Agent** (`heartbeat_agent.py`) - Zero-dependency Python script that sends periodic pings
2. **Azure Function** (`Ping/`) - HTTP endpoint that receives and logs pings
3. **Azure Monitor Alert** - Scheduled query rule that detects missing pings (5-minute window)
4. **Action Group** - Email notification system

## Target Users

Home users and small businesses monitoring internet connectivity on specific devices (e.g., home servers, IoT devices).
